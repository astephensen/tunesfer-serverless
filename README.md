The serverless side of the Tunesfer web interface.

## Setup

There are two environment variables you must populate with your Spotify API key. This can be stored in a `.env` file for convenience.

* `SPOTIFY_CLIENT_ID`
* `SPOTIFY_CLIENT_SECRET`
